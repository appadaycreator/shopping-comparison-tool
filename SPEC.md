# お得な買い物比較ツール【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Shopping Comparison Tool
**バージョン**: 2.2.1（M8入力ガイダンス強化・FAQPageスキーマ追加版）
**更新日**: 2026-06-14
**URL**: https://appadaycreator.com/shopping-comparison-tool/

Amazon・楽天・Yahoo等の価格とポイント還元を比較して実質最安値を計算。登録不要・完全無料でご利用いただけます。

## v2.2.1 の改善内容（2026-06-14）

### SEO・ユーザーガイダンス強化
- **M8 入力ガイダンス大幅充実**: 各入力欄の「?」ツールチップに詳細な説明を追加。価格入力時の注意点（税抜き/税込み）、ポイント還元率の相場情報（楽天1～5%、Amazon0.5～2%）、送料の確認方法を明示。タイトル属性でも追加ヒント表示。
- **FAQPageスキーマ追加**: 5つのよくある質問にFAQPageスキーマを付与。最安値の見つけ方、ポイント還元計算、無料利用、データ保存、モバイル対応を網羅。GSC表示回数ゼロ対策。
- **メタタイトル・description強化**: 「価格比較・ポイント還元計算Webツール」と「複数EC間の送料込みコスト比較で買い物を節約」のキーワード追記。検索流入改善狙い。

## v2.2.0 の改善内容（2026-06-14）

### 新機能
- **目標設定・達成度トラッキング（M6）**: 目標節約額をlocalStorageで保存。比較結果から自動計算した節約額で達成度を表示。プログレスバーで視覚的に目標達成状況を確認可能。継続利用を促進。
- **印刷・結果エクスポート機能（M10）**: 結果をクリップボードにコピー、CSVダウンロード、ブラウザ印刷の3つのエクスポート機能を実装。ユーザーが比較結果を活用・保存しやすく。
- **入力ガイダンス充実（M8）**: 各入力欄に「?」ツールチップを配置。価格入力の注意点、ポイント還元率の相場情報、送料の初期値設定理由を明示。

## v2.1.0 の改善内容（2026-06-10）

### 新機能
- **データの自動保存・復元（M3）**: localStorage を使用した入力値の自動保存。ページ再読み込み時に前回のデータを自動復元。復元バナーで確認後、ワンクリックで前回データを復元可能。
- **詳細なガイダンス・ツールチップ（M8）**: 各入力欄に「?」アイコンを追加。ポイント還元率の相場、送料の初期値設定理由など具体的な説明をツールチップで表示。新規ユーザーが迷わずに入力できるよう改善。
- **データ保存状況の表示**: ツール上部に「入力データは自動保存されています」メッセージを表示。ローカルストレージ使用・外部送信なしを明示。

## v2.0.0 の改善内容（2026-06-03）

### 新機能
- **複数商品の同時比較**: 「商品を追加」ボタンで複数商品の入力フォームを動的に生成・管理
- **送料・税金対応**: 各店舗の送料を入力可能。税込/税抜を選択可能
- **比較結果のグラフ表示**: Chart.js で実質価格を棒グラフで可視化
- **節約額の自動計算**: 複数店舗の価格差を「最大○○円の節約」で表示

### UI/UX 改善
- モバイル最適化：3列グリッドを1列に自動切り替え
- 削除ボタン：各商品欄に「削除」ボタン追加
- デフォルト送料：楽天・Yahoo は送料 500 円でプリセット

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: お得な買い物比較ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
